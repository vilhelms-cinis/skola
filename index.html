Kam izmanto <div> tagu?

Mēs izmantojam <div>, lai atdalītu vai grupētu elementus HTML dokumentā - tas bieži ir konteineris citiem elementiem, lai tos būtu vieglāk stilizēt ar CSS vai arī manipulēt ar Javascript.

Kam izmanto <span> tagu?

Šo tagu mēs izmantojam, ja vēlamies grupēt vairākus elementus, kas atrodas vienā rindiņā, atšķirībā no <div> taga, kas var grupēt veselus blokus. Tas nerada nekādas vizuālas izmaiņas (atkal atšķirība no <div>), kas redzami atdala elementu bloku), tomēr var kalpot, lai pievienotu CSS dizainu vai Javascript kodu.
